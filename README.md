# Site para Consultório Odontológico

Projeto de desenvolvimento de um site institucional para consultório odontológico, com o objetivo de apresentar os serviços oferecidos, facilitar o contato com pacientes e melhorar a presença digital do consultório.

## Funcionalidades

- Página inicial com apresentação do consultório
- Seção de serviços odontológicos
- Informações de contato
- Layout simples e organizado para facilitar a navegação

## Tecnologias utilizadas

HTML  
CSS  
JavaScript

## Objetivo do projeto

Este projeto foi desenvolvido com o objetivo de praticar:

- desenvolvimento de páginas web
- estruturação de layouts com HTML e CSS
- criação de interfaces simples para apresentação de serviços

## Como visualizar o projeto

1. Baixe ou clone o repositório
2. Abra o arquivo `index.html` no navegador
